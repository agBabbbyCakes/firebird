Minimal Elemental Design

A lightweight, semantic CSS system. No complex classes, just HTML.

View Components How to Use

1. How to use

Simply link the stylesheet in your HTML head. No build steps required.

<link rel="stylesheet" href="/assets/minimal.css" />

2. Global Customization

Change the look globally by editing the :root variables in assets/minimal.css.

Colors

--bg
--accent
--border

Edit these to change the theme instantly.

Layout

--width: 1200px (Max site width)
--spacing: 2rem (Global padding)
--radius: 8px (Rounded corners)

Controls spacing and geometry.

3. Typography

Headings and text style automatically based on semantic tags.

Heading 1

Heading 2

Heading 3

Heading 4

Body text paragraph. Uses --text-muted color by default for better readability on dark backgrounds.

Code Example

<h1>Page Title</h1>
<h2>Section Title</h2>
<p>Content goes here...</p>

4. Components

Interactive elements and layout blocks.

Buttons

Uses <button> or <a role="button">

Cards (Article)

Use the <article> tag to create cards automatically.

Grid Layouts

Use .grid for standard content or .grid-compact for smaller items.
