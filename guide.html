Global Customization

Modify assets/minimal.css to change the theme instantly across the entire site.

Variables (:root)

:root {
  --bg: #050608;          /* Background Color */
  --text: #e5e7eb;        /* Main Text Color */
  --accent: #00FFFF;      /* Brand Accent */
  --font-body: system-ui; /* Font Family */
  --radius: 8px;          /* Border Radius */
  --spacing: 2rem;        /* Spacing Unit */
  --width: 1200px;        /* Max Content Width */
}

Change --accent to update all buttons and links. Change --radius to make cards more square or round.
